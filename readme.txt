== Description ==

De 'weTickets e-ticket shop' is een plugin die het gemakkelijk maakt uw eigen ticket shop toe te voegen aan uw blog post. Wilt u ook gemakkelijk uw tickets verkopen via internet en heeft u nog geen account? Ga dan naar http://wetickets.nl en maak een gratis account aan! 

== Installation ==

1. Download [wetickets plugin](http://downloads.wordpress.org/plugin/embed-wetickets.zip)
2. Uitpakken
3. Kopieer naar folder '/wp-content/plugins'
4. Activeer plugin

== Usage ==

Gebruik de volgende tag om de e-ticket shop in te voegen bij uw post

`[wetickets shopID]` of `[wetickets shopID breedte]`

bijv.

`[wetickets 44]` of `[wetickets 44 500]`
